# Arithmetic #

Arithmetic is fundamental to computation.
`Whamm` defines a number of arithmetic operators on the primitive types.

## Integer types ##

```
var a: i32;
a = 0;
a++;       // increment == 1
a--;       // decrement == 0
a = 9 + 3; // add == 12
a = 8 - 2; // subtract == 6
a = 7 * 4; // multiply == 28
a = 9 / 2; // divide == 4
a = 5 % 3; // modulus == 2
```
